## How to Deploy on GitHub Pages

1.  **Create a New Repository:**
    - Go to [GitHub](https://github.com) and create a new repository (e.g., `my-valentine`).
    - Make sure it's **Public**.

2.  **Push the Code:**
    - Open your terminal in this project folder.
    - Run the following commands:
      ```bash
      git init
      git add .
      git commit -m "Initial commit for my valentine"
      git branch -M main
      git remote add origin https://github.com/YOUR_USERNAME/my-valentine.git
      git push -u origin main
      ```
    *(Replace `YOUR_USERNAME` and `my-valentine` with your actual username and repo name)*

3.  **Enable GitHub Pages:**
    - Go to your repository **Settings**.
    - Scroll down to the **Pages** section (or click "Pages" in the left sidebar).
    - Under **Source**, select `Deploy from a branch`.
    - Under **Branch**, select `main` and folder `/ (root)`.
    - Click **Save**.

4.  **Send the Link:**
    - Wait a minute or two.
    - Refresh the Pages settings page to see your live URL (e.g., `https://your-username.github.io/my-valentine/`).
